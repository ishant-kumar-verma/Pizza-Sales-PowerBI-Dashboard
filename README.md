# 🍕 Pizza Sales Analytics Dashboard

## 📌 Overview

This is an interactive Power BI dashboard developed using the Maven Pizza Sales dataset. The dashboard provides insights into sales performance, revenue trends, customer ordering patterns, and pizza performance.

---

## 🛠️ Tools Used

- Power BI
- DAX
- Power Query
- SQL
- Excel
